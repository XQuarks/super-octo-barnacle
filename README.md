# 超级八爪藤壶 · super-octo-barnacle

零后端 AI 互动叙事引擎 — 输入任意世界观，即刻生成沉浸式文字冒险。支持多规则集（D&D/克苏鲁/科幻/现代/纯叙事），纯前端，GitHub Pages 一键部署。

---

## 快速开始

```bash
python -m http.server 8000
# 打开 http://localhost:8000
```

> 不要用 `file://` 打开，安全策略会阻止加载数据文件。

**部署**：Push 到 GitHub → Settings → Pages → Source 选 `main` 分支，目录 `/ (root)` → Save。纯静态文件，无需构建。语义搜索模型（22MB）已内嵌，首次加载后缓存 IndexedDB，之后秒开。

## 使用流程

1. 首页点击「开始冒险」→ 创建新世界
2. **页签式创建**：世界设定 → 规则设置 → 风格选项 → 高级选项
3. **选择规则集**：D&D 奇幻 / 克苏鲁恐怖 / 科幻 / 现代 / 纯叙事 / AI 自动生成
4. 填入 DeepSeek API Key → AI 生成知识库、开场白和初始选项
5. **角色创建**：选择种族职业 → 属性投骰 → 确认角色
6. 进入游戏：自由输入或点选选项 + CRPG 动作菜单，AI 每轮生成叙事 + 状态变更

关闭「模拟模式」配置 API：

| 配置项 | 默认值 |
|---|---|
| Base URL | `https://api.deepseek.com` |
| API Key | 你的密钥 |
| 模型 | `deepseek-v4-flash` |

> **当前版本仅针对 DeepSeek 模型优化**（前缀缓存、token 配额控制），使用其他模型可能出现 token 消耗过大。

## 核心特性

### 叙事引擎
- AI 生成完整世界配置：知识库、角色、开场白、初始选项
- 自适应叙事基调：自动识别日常/悬疑/恋爱/高张力题材，调整节奏
- 主角硬约束：设定不会被 AI 遗忘或降级
- RAG 混合检索：中文分词 + 向量语义排序（内嵌 ONNX 模型，无需网络）
- 流式打字机：SSE 流式输出 + 逐字渲染
- 死亡存档：红色标记，可回顾全程

### CRPG 系统
- **多规则集支持**：D&D（D20/种族职业）、克苏鲁（D100/理智值）、科幻/现代（技能检定）、纯叙事（无骰子）
- **规则自由度**：1-5 级，从"骰子严格决定一切"到"纯叙事驱动"
- **世界观自由度**：1-5 级，从"严格遵循源材料"到"仅借用世界观框架"
- **角色创建**：种族选择 → 职业选择 → 属性投骰（4d6取最高3）
- **D20 战斗系统**：回合制战斗、HP/MP/AC、敌人数据、战斗面板
- **瓦片地图**：像素风格可交互地图，显示地形/实体/NPC
- **动作菜单**：攻击/施法/潜行/说服/调查/生存（根据规则集动态调整）

### 界面
- 上下分区：上方地图/战斗可视化交互区 + 下方剧情滑动区
- 拖拽分隔线调整上下比例
- 角色状态面板：属性/背景/关系/物品/技能/目标
- 暗色/白天双主题 + 字体缩放

## 项目结构

```
├── index.html              # 唯一入口，含全部CSS
├── js/
│   ├── app-core.js         # 全局状态/常量/主题
│   ├── app-ui.js           # 初始化/UI渲染/世界生成
│   ├── app-ai.js           # RAG/System Prompt/LLM/安全
│   ├── app-game.js         # 游戏循环/状态变更/战斗/存档
│   ├── dice.js             # D20/D100 骰子系统
│   ├── combat-stats.js     # 战斗数值管理
│   ├── character.js        # 角色创建数据引擎
│   ├── combat-engine.js    # 回合制战斗引擎
│   ├── action-menu.js      # CRPG 动作菜单
│   ├── map-data.js         # 瓦片地图数据解析
│   ├── renderer/tile-map.js # 瓦片地图渲染
│   └── ui/character-creator.js # 角色创建 UI 向导
├── data/
│   ├── rulesets.json       # 6 套规则集定义
│   ├── character_creation.json # 种族/职业元数据
│   ├── enemy_data.json     # 敌人模板
│   ├── lore_kb.json        # 示例知识库
│   ├── lore_kb_with_embeddings.json # 预计算向量
│   ├── initial_state.json  # 默认初始状态
│   └── system_prompt_template.md # Prompt 模板
├── models/                 # ONNX 语义搜索模型 (22MB)
└── tools/                  # 开发工具
```
